SEGUINDO ESSES PASSOS VOCÊ INICIA UM PROJETO REACT.JS

cd + diretório // // ex -> C:\Users\marce\OneDrive\Documentos\VscodeProjetos\Estudos Gerais\Front-end
para especificar onde quer incluir seu projeto, você pode copiar caminho lá no arquivo

npm create vite@latest nomeProjeto --template react // para criar um projeto vite + react

npm install para aparecer o node_modules
npm install -D sass para baixar o sass
npm install react-icons para instalar os icons usado no react icons
npm install react-router-dom para baixar o react routes
npm install axios para baixar bibloteca de chamadas de API (com o back-end e afins)

code . // para entrar no projeto no vscode // atalho para entrar no projeto pelo vscode rapidamente

NO PROJETO: 
obs: se não tiver a pasta "node_modules" precisa baixar o npm para designar um local host para o projeto, logo no terminal escreva:
npm install

em seguida:
npm run dev // para ser dito em que local host está seu projeto e fornecer o link
